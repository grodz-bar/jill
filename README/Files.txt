DIRECTORY STRUCTURE:
-------------------
The bot follows this directory structure:

    /jill/
    ├── bot.py                            # Main bot file
    ├── requirements.txt                  # Python dependencies
    ├── .env                              # Bot configuration (created during setup)
    ├── /config/                          # Configuration folder (USER CUSTOMIZATION)
    │   ├── __init__.py                   # Python package marker
    │   ├── aliases.py                    # Command aliases
    │   ├── features.py                   # Feature toggles
    │   ├── messages.py                   # Bot messages and responses
    │   ├── timing.py                     # Timing and cooldown settings
    │   └── paths.py                      # File paths configuration
    ├── /handlers/                        # Command handlers (implementation)
    │   ├── __init__.py                   # Python package marker
    │   └── commands.py                   # All bot commands
    ├── /core/                            # Core music player (implementation)
    │   ├── __init__.py                   # Python package marker
    │   ├── player.py                     # MusicPlayer class
    │   ├── playback.py                   # Playback functions
    │   └── track.py                      # Track class and library loading
    ├── /systems/                         # Specialized systems (implementation)
    │   ├── __init__.py                   # Python package marker
    │   ├── spam_protection.py            # 5-layer spam protection
    │   ├── cleanup.py                    # Message cleanup (TTL + history)
    │   ├── voice_manager.py              # Voice operations & auto-pause
    │   └── watchdog.py                   # Playback monitoring
    ├── /utils/                           # Utility helpers (implementation)
    │   ├── __init__.py                   # Python package marker
    │   ├── discord_helpers.py            # Discord API helpers
    │   ├── persistence.py                # Channel persistence
    │   └── context_managers.py           # Context helpers
    ├── /README/                          # Documentation folder
    │   ├── README.txt                    # Main overview
    │   ├── SETUP-Linux.txt               # Linux setup guide
    │   ├── SETUP-Windows.txt             # Windows setup guide
    │   ├── Getting-Discord-Token.txt     # Bot token guide
    │   ├── Converting-To-Opus.txt        # Audio conversion guide
    │   ├── troubleshooting.txt           # Troubleshooting guide
    │   └── Files.txt                     # This file
    └── /music/                           # Your music files go here
        ├── 01 - Track Name.opus
        ├── 02 - Track Name.opus
        └── ...
        │ # OR if you have multiple playlists:
        ├── 01 - Album Name/
        │   ├── 01 - Track Name.opus
        │   └── 02 - Track Name.opus
        ├── 02 - OST/
        │   └── 01 - Track Name.opus
        └── ...


FILES DESCRIPTION:

README.txt
----------
Main overview - start here! Quick feature list, commands, and basic info.

SETUP-Linux.txt
----------------
Detailed setup guide for Linux/Pi.

SETUP-Windows.txt
------------------
Detailed setup guide for Windows.

Getting-Discord-Token.txt
-------------------------
Quick guide for creating a Discord bot and getting your token.

Converting-To-Opus.txt
-----------------------
Instructions for converting your music to .opus format. (Windows/Linux)

troubleshooting.txt
-------------------
Troubleshooting guide for common issues.

.env.example
------------
Sample environment configuration file.
Copy or rename to ".env" and fill in your bot token and music folder path.

requirements.txt
----------------
Python package dependencies for easy installation.
Use with: pip install -r requirements.txt

/config/ (Configuration Folder)
------------------------------
Configuration files for easy customization:
- config/aliases.py    - Command aliases customization.
- config/features.py   - Feature toggles (turn features on/off)
- config/messages.py   - Customize responses, messages, and drink emojis.
- config/timing.py     - Timing and cooldown settings. (Advanced)
- config/paths.py      - File paths and storage location. (Advanced)

last_channels.json
------------------
Channel persistence storage file (auto-created).
Stores the last used text channel per Discord server.
Allows cleanup features to resume after bot restart.
Automatically managed - do not edit manually.

.gitignore
----------
Tells git which files to ignore (logs, tokens, music, runtime state, caches, etc.)
Prevents accidentally committing sensitive data and temporary files.

================================================================================
SETUP ORDER (Recommended):
================================================================================

1. Read 'README.txt' (get familiar with the bot)
2. Follow 'Getting-Discord-Token.txt' (get your bot token)
3. Follow 'Converting-To-Opus.txt' (prepare your music)
4. Follow 'SETUP-Linux.txt' or 'SETUP-Windows.txt' (install and configure the bot)
5. Run the bot!
